Day la chuong trinh giup tao thong tin chinh chu cho folder.

Khi chay chuong trinh se tao ra cau truc nhu sau:
-auth
---auth.txt
---log.txt

Trong do file auth gom nhung thong tin sau:
Name: ten project
Version
Author
Date created
Date fixed

File log de ghi chep lai nhung thay doi qua tung Version.

Co 2 tuy chon la new va old. New se tao moi hoan toan thu muc auth, old se dua vao version cu de cap nhat lai va ghi thong tin cap nhat vao log.txt.
